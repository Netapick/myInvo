===============================================
               myInvo - Guide d'utilisation
               Version 1.4 avec système de licence sécurisé
===============================================

INSTALLATION
============
1. Exécutez le fichier d'installation myInvo-1.4-Secure-License-Setup.exe
2. Suivez les instructions à l'écran
3. L'application sera installée dans le dossier Programme Files
4. Un raccourci sera créé sur le bureau (optionnel)

PREMIÈRE UTILISATION
===================
1. Lancez myInvo depuis le menu Démarrer ou le raccourci bureau
2. Au premier démarrage, l'application vous proposera d'activer une licence
3. Vous pouvez :
   - Utiliser le mode d'évaluation (limité dans le temps)
   - Activer une clé de licence complète

ACTIVATION D'UNE LICENCE
=======================
1. Dans myInvo, allez dans le menu "Licence" ou "Activation"
2. Saisissez votre clé de licence au format : XXXX-XXXX-XXXX-XXXX-XXXX
3. Les informations (nom, entreprise) sont automatiquement extraites de la clé
4. Cliquez sur "Activer" pour valider votre licence

TYPES DE LICENCES
================
- Trial (Évaluation) : 30 jours d'utilisation complète
- Standard : 1 an d'utilisation avec toutes les fonctionnalités
- Premium : Licence permanente avec support prioritaire

FONCTIONNALITÉS PRINCIPALES
===========================
✓ Création de devis et factures
✓ Export PDF professionnel
✓ Archivage automatique
✓ Calculs TVA et totaux


DOSSIERS CRÉÉS
==============
- config/     : Configuration et informations utilisateur
- devis/      : Sauvegarde des devis
- factures/   : Sauvegarde des factures
- archives/   : Archives automatiques
- logs/       : Fichiers de journalisation

SUPPORT TECHNIQUE
================
En cas de problème :
1. Vérifiez les logs dans le dossier logs/
2. Consultez la licence dans le menu "À propos"
3. Contactez le support avec votre numéro de licence
